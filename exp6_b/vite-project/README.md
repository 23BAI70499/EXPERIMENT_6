## 🚀 Project

Client-Side Form Validation (Sub Exp-2) using React (Vite)

### ✨ Features

* Controlled Components (Email & Password)
* Client-side validation before submission
* Email format validation (@ and valid domain)
* Password strength rules enforcement
* Error messages display
* Form submit only when valid
* Clean Dark UI

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* CSS

---

## ▶️ Run Locally

```bash
npm install
npm run dev
