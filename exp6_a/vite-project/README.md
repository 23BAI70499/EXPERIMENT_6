## 🚀 Project

Controlled Form using React (Vite)

### ✨ Features

* Controlled Components
* Auto Age Calculation from DOB
* Radio, Checkbox, Dropdown handling
* Form Reset (Cancel)
* Clean Dark UI

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* CSS

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```
