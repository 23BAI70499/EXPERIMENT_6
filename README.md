# 🚀 EXP6 - React Form Validation Projects

This repository contains two React-based applications demonstrating different approaches to form handling and validation.

---

## 📁 Projects Included

### 1️⃣ Client-Side Form (clientsideform/)

A React application focused on real-time client-side validation with interactive feedback.

#### ✨ Features
- 📧 Email Validation (valid format + .com domain)
- 🔐 Strong Password Rules (capital, lowercase, number, special char, 8+ length)
- 📊 Password Strength Indicator (Weak / Moderate / Strong)
- 🎨 Real-time UI feedback and alerts

#### 🛠 Tech Stack
React 18, Vite, CSS3, JavaScript (ES6+)

---

### 2️⃣ Submission Form (submissionform/)

A comprehensive profile form handling multiple input types.

#### ✨ Features
- 👤 Personal Info (Name, DOB, Gender)
- 🧠 Skills Selection (Programming, Web, DB, AI/ML)
- 📍 Address Input + State Dropdown
- ⚙️ Submit & Reset Controls
- ✅ Built-in HTML5 Validation

#### 🛠 Tech Stack
React 18, Vite, CSS3, JavaScript (ES6+), useState, useMemo

---

## ⚙️ Getting Started

### Prerequisites
- Node.js (v16+)
- npm / yarn

### ▶️ Run Client-Side Form
cd clientsideform
npm install
npm run dev

### ▶️ Run Submission Form
cd submissionform
npm install
npm run dev

App runs at: http://localhost:5173
